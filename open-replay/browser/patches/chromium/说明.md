# Chromium Fork 端补丁集

> 应用到 Replay.io 的开源 chromium fork 上让它能在现代 macOS（15.6 + Xcode 15.1 + Python 3.13）上构建的修补集合。
>
> Replay.io chromium fork：`https://github.com/replayio/chromium`
> 我们用的 commit：`master` 分支（截至 2026-04-27 末）

## 怎么用

```bash
# 1. clone Replay.io chromium fork
git clone https://github.com/replayio/chromium.git /path/to/chromium
cd /path/to/chromium

# 2. 给 v8 子目录配 DEPS 钉的 commit（DEPS 里 v8_revision 就是这个）
cd src/v8
git remote -v  # 应当指向 github.com/replayio/chromium-v8.git
# 如果不是，参考 困难记录.md #18 修

# 3. 应用补丁
cd /path/to/chromium/src
patch -p1 < /path/to/open-replay/browser/patches/chromium/all-build-fixes.patch

# 4. 拷贝两个新建的 package.json
cp /path/to/open-replay/browser/patches/chromium/new-files/package.json .
cp /path/to/open-replay/browser/patches/chromium/new-files/chrome/browser/resources/tools/package.json \
   chrome/browser/resources/tools/

# 5. 替换 third_party/six/src/six.py（chromium 自带 1.14.0 在 Python 3.12+ 已坏）
pip3 install --user --break-system-packages six  # 装到 user-site
cp ~/Library/Python/3.13/lib/python/site-packages/six.py third_party/six/src/six.py

# 6. 现在可以 gn gen + ninja chrome 了
```

## 子仓库补丁

`third_party/swiftshader/` 是单独的 git 子仓（不归 chromium 主仓 commit），所以它的修补单独打成 `swiftshader-llvmjit-arm64.patch`。应用：
```bash
cd /path/to/chromium/src/third_party/swiftshader
patch -p1 < /path/to/open-replay/browser/patches/chromium/swiftshader-llvmjit-arm64.patch
```
内容：在 `src/Reactor/LLVMJIT.cpp` 把 `__sincosf_stret` 限定到 x86 macOS（arm64 SDK 14.2 已不导出该符号），arm64 用 `sincosf` 兜底。

## TypeScript node_modules 修补

`third_party/node/node_modules/typescript/lib/lib.dom.d.ts` 不在 chromium git 里，所以走 shell 脚本：
```bash
bash typescript-trusted-types.sh /path/to/chromium/src
```
内容：在 lib.dom.d.ts 末尾追加 TrustedHTML / TrustedScript / TrustedScriptURL 占位 type alias，绕开 tsc 5.0.3 的"引用未声明类型"问题。

## 不在 patch 里的需要手工做的事

1. **替换 `third_party/six/src/six.py`** —— bundled 1.14.0 在 Python 3.12+ 上 `six.moves` 全坏。装系统 six 1.17.0 然后覆盖。详见 困难记录 #23。
2. **`out/Release/args.gn`** —— 见 `路线图.md` Phase B1'.3 的目标 args.gn。
3. **如果 `chromium/src/v8/` 里是 vanilla 上游 V8（没有 `replayio-macros.h`）**：要把它换成 `github.com/replayio/chromium-v8.git@<DEPS 钉的 commit>`。详见 困难记录 #18。
4. **运行 `bash typescript-trusted-types.sh`**（见上）
