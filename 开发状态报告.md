## 使用流程

### 1. 安装和配置
```bash
cd v8-recorder
./setup.sh      # 下载 V8 并应用补丁
./build.sh      # 编译 V8 (30-60 分钟)
```

### 2. 编译示例
```bash
./compile-examples.sh
```

### 3. 录制执行
```bash
./simple        # 生成 simple.rec
./fibonacci     # 生成 fibonacci.rec
```

### 4. 分析录制
```bash
python3 analyze.py fibonacci.rec
python3 analyze.py fibonacci.rec --trace
```

### 5. 重放调试
```bash
./replay fibonacci.rec
# 选择模式: 1=完整重放, 2=单步, 3=断点
```
