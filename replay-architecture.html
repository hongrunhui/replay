1. 产品概述

Replay.io 是一个时间旅行调试平台（Time-Travel Debugging Platform），它能够完整记录浏览器中的 JavaScript 执行过程，并支持确定性重放（Deterministic Replay）。与传统的会话重放工具（Session Replay）不同，Replay.io 不仅记录 DOM 变化，而是记录整个运行时（Runtime）的执行过程。

🎯

确定性重放

记录后可以精确重现程序执行过程

⏰

时间旅行调试

可以在任意时间点暂停、检查和回溯

📦

完整运行时捕获

记录所有 JS 执行、DOM 操作、网络请求

⚡

低开销

录制开销约 3.5%，低于传统工具

7. 关键环境变量

🎬

RECORD_ALL_CONTENT

值: "1" - 启用完整内容录制，浏览器会记录所有 JS 执行、DOM 操作、网络请求

📁

RECORD_REPLAY_DIRECTORY

值: "~/.replay" - 指定录制文件存储位置，浏览器将数据写入此目录

📝

RECORD_REPLAY_METADATA

值: JSON 字符串 - 传递元数据（processGroupId、title、uri）给浏览器

🔇

RECORD_REPLAY_DONT_RECORD

值: "1" - 禁用录制，用于在 Replay 浏览器中正常浏览而不录制
